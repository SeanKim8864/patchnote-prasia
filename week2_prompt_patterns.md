## 결사전 참가 주기 및 방식

### 📌 유니버스리그 구조 (현행 기준)

```
1개 시즌 = 라운드 1 → 라운드 2 → 라운드 3 (약 3주)
```

각 라운드는 **수요일(참가 신청)** → **목요일~화요일(라운드 진행)** 구조입니다.

| 단계 | 시간 |
|------|------|
| 참가 신청 및 팀 편성 | 수요일 낮 12시 ~ 오후 11시 59분 |
| 라운드 진행 | 목요일 오후 6시 ~ 다음 화요일 오후 11시 59분 |
| 다음 라운드 반복 | 3라운드 완료 후 시즌 종료 |

---

### 🗓️ 라운드 내 주간 일정

| 요일 | 콘텐츠 | 시간 |
|------|--------|------|
| **일요일** | 거점전 선포 | 오후 9:50 ~ 10:30 (40분) |
| | 거점전 진행 | 선포 후 최대 오후 11:10까지 |
| **목요일** | 보스전 → 일요일로 통합됨 (2026.02.11~) | — |
| 목/일 | 피크타임 (유니버스리그 점수 활성도 반영) | 목 오후 9~11시 / 일 오후 8~11시 |

### 🏰 거점전 선포 방식 (리네아 일반 거점)

**조건**
- 결사 레벨 3 이상
- 결사장(또는 권한 위임자)이 에테르 상자를 소모하여 선포
- 렐름 이전 후 일정 시간(일반 24시간 / 인피니티 시즌 한정 1시간) 경과 필요
- 거점전 종료 직후 5분간 보호 상태 → 선포 불가

**선포 비용**

| 거점 종류 | 선포 비용 |
|-----------|-----------|
| 요새 | 텍스토르 에테르 50개 (시즌2 이후 200→50으로 인하) |
| 성채 | 별도 상이 |
| 본부 레벨 25 이상 | 비용 변동 없음 (고정) |

---

### ⚔️ 유니버스리그 거점전 참가 방식

**① 팀 구성**
- 결사장이 참가 신청 → 다른 결사를 초대해 팀 편성
- 팀 탈퇴·추방은 라운드 종료 1시간 전까지만 가능 (기존 24시간에서 단축)
- 팀 참여 중에는 참가 신청 취소 불가

**② 리그 배정**
- 라운드1: 팀 전력 기반(결사원 전투력 → 가중치 적용)으로 리그 결정
- 라운드2~3: 직전 라운드 유니버스리그 점수 기준으로 승강 결정

**③ 리그 종류 (팀 수)**

| 리그 | 팀 수 |
|------|-------|
| 챌린저 | 4팀 |
| 마스터 | 6팀 |
| 플래티넘 | 16팀 |
| 실버 | 34팀 |

---

### 📊 점수 체계 (유니버스리그)

| 점수 종류 | 비중 | 획득 방법 |
|-----------|------|-----------|
| **리그별 가중 점수** | 높음 | 배정된 리그·매치에 따른 가중치 자동 적용 |
| **거점 점령 점수** | 높음 | 거점전 종료 시 텍스토르 내 점령 거점에 따라 차등 지급 |
| **보스전 점수** | 중간 | 보스 처치 기여도 기반, 순위별 점수 치환 |
| **활성도 점수** | 낮음 | 거점전 승/패 결과 + 피크타임 입장 인원 + 이용 시간 (최대 제한 있음) |

**거점 순위 (점수 비례)**
1위 빛무리 성채 → 2위 물바위숲 요새 → 3위 불바위숲 요새 → 4~7위 주둔지 4곳
